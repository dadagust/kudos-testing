# Задача: сверстать хедер + hero-блок лендинга KUDOS

## Контекст

Проект: фронтенд лендинга сервиса аренды мебели и декора KUDOS.  
Стек: Next.js 14, React, TypeScript, CSS Modules (.module.sass).  
Нужно сверстать верхнюю часть лендинга (верхняя контактная плашка + хедер с меню + hero-блок с фоновым фото и кнопкой) строго по дизайну из Figma.

- Цвет счетчика на корзине: `#4A4B4D` текст на `#4A4B4D` кружке (заменить на реальные)

Все, что выходит за эти рамки просто будет пусто

## Требования к вёрстке

1. **Структура блока**
   - Верхняя серая плашка с:
     - адресом слева (`[pin icon] МО, Люберцы, квартал 30131, 1020`);
     - email и телефоном справа (`[mail icon] info@kudos.ru`, `[phone icon] +7 (495) 991-05-79`).
   - Основной хедер:
     - слева кнопка "Каталог" с иконкой грид-меню;
     - по центру логотип KUDOS;
     - справа навигация ("Доставка и самовывоз", "Условия работы", "Вопросы и ответы") и блок иконок `[heart] [cart с badge «2»] [user]`.
   - Под хедером — полноэкранный hero c фоновой картинкой `docs/design/kudos-hero-desktop.png` (через `next/image` с `fill` и `object-fit: cover`).
   - В левой нижней части hero:
     - текст "АРЕНДА МЕБЕЛИ И ДЕКОРА" в две строки;
     - под ним кнопка "Подробнее".

2. **Пиксель-перфект**
   - Все размеры, отступы и выравнивания взять из Figma.
   - Ширина контейнера, межколоночные расстояния и высота хедера должны совпадать с фреймом 1440 в Figma.
   - Логотип и иконки должны быть выровнены по вертикали с текстом как на макете.

3. **Кнопка "Подробнее"**
   - Использовать общий компонент `Button` из `src/shared/ui/button/Button.tsx` с пропсами:
     - `variant="primary"`, `size="lg"` (пример — заменить реальными).
   - Стиклинг кнопки (цвет фона, бордер, ховер) строго по Figma.

4. **Адаптив**
   - На этом этапе сверстать только desktop-вариант (ширина 1440).  
     Адаптив будет отдельной задачей.

## Проверка

- При ширине окна 1400px скриншот страницы должен визуально совпадать с `docs/design/kudos-hero-desktop.png`.
- Нет горизонтального скролла.
- Верстка валидна, без ошибок линтеров (ESLint, Stylelint).
